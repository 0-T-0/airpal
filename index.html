Airpal

A web-based query execution tool built on top of Facebook's PrestoDB.

Get Started

Facilitate Analysis

No more copying and pasting in the command line interface. Airpal reduces the friction involved in data analysis by making it easy to find tables, run queries, save analysis, and get results on your desktop.

Drive SQL Adoption

At organizations with different levels of analysis sophistication, Airpal helps make it easy for beginners to explore datasets and write queries.

Manage Access

Airpal can act as a security layer on top of Presto, where administrators can designate certain tables as accessible for certain groups of users.
